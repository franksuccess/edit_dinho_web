Software de Trabajo

ADOBE PREMIERE PRO

Mi motor principal para montaje profesional. Lo utilizo para estructuras narrativas complejas y edición de alta precisión.

AFTER EFFECTS

El estándar para motion graphics y efectos visuales avanzados (VFX). Aquí creo las animaciones que hacen resaltar cada edit.

DAVINCI RESOLVE

Herramienta líder para corrección de color profesional. Aseguro que cada clip tenga una estética cinematográfica única.

CAPCUT (Desktop/Mobile)

Ideal para ediciones rápidas y tendencias virales de TikTok. Aprovecho su agilidad para contenido vertical de alta rotación.

FILMORA

Software versátil que utilizo para layouts creativos y recursos rápidos cuando el proyecto requiere un estilo fresco y moderno.

SONY VEGAS PRO

Un clásico para edición de audio y video con un flujo de trabajo intuitivo, excelente para sincronización rítmica (AMVs).
